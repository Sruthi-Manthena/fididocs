# DotSight

DotSight, an advanced data pipeline management tool, offers unparalleled customization for portfolio analytics in the on-chain web3 space. Users can effortlessly create and manage tailored data pipelines, enjoying the flexibility of defining sources, implementing potent mixers and mappers, and executing operations with a simple GET request. The platform's intuitive design empowers users to monitor their crypto portfolios, track financial metrics, and access real-time data seamlessly. DotSight's integration capability allows protocols to contribute to its library, expanding the horizon of data possibilities and marking a significant stride in the data revolution.
