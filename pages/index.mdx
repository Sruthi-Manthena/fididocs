# Overview
 
FiDi is a hub for advanced high-precision analytical data in Web3.

FiDi Portfolio Analytics is a presentation layer with a breadth of 100+ chains/2.5k+ protocols,  offering unparalleled analytical depth for selected networks, and support for all major wallet providers. 

DotSight is an actively evolving platform for developers. 
It enables you to build highly customizable data pipelines for on-chain Web3 data. The no-code UI and user-defined views are coming soon. Visit our Git repo to test it out and contribute.

FiDi's Data API is also currently available for businesses. 
Drop us a note here to learn about current and custom integration options.

Stay tuned and follow our Twitter for more updates.
